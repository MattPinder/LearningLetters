# Learning Letters

A prototype app for teaching learners how to correctly write the letters (upper- and lower-case) used in English.

As this app is still a prototype, constructive feedback is always appreciated!

## How to install

1. Go to the `Builds` folder, click on the latest APK file (currently `LearningLetters_Build8_20250318.apk`),  then download it onto your mobile device by clicking the `...` icon and `Download`.

![DownloadFile](https://raw.githubusercontent.com/MattPinder/LearningLetters/main/.github/images/DownloadFile.jpg)

2. Open the downloaded APK file, which should install Learning Letters on your device.

**Note:** As this is not an official App Store app, you may need to either click OK/Proceed if the device asks you to confirm if you want to install, or adjust your device's settings to allow installation from APK files.

## How to play

Once the app is installed and opened, instructions can be accessed via the Instructions button.
